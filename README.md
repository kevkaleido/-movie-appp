## Getting Started

To get started with the project, follow these steps:

1. **Install dependencies**:
    ```sh
    npm install
    ```

2. **Run the application**:
    ```sh
    npm start
    ```

3. **Build the application for production**:
    ```sh
    npm run build
    ```

4. **Run tests**:
    ```sh
    npm test
    ```
